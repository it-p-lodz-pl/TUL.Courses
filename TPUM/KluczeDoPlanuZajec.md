# Klucze do samodzielnego zapisu

Proszę przekazać pocztą pantoflową klucze do samodzielnego zapisu puki to jeszcze możliwe.

| Plan                 |   Klucz   |
| -------------------- | :-------: |
| Środa nieparzysta    | **`TP1`** |
| Czwartek nieparzysty | **`TP2`** |
| Środa parzysta       | **`TP3`** |
| Czwartek parzysty    | **`TP4`** |
